#### User Profiles
Every wallet address automatically gets a user profile with:

- **Follow Users**: Connect with top traders and active community members

## Using Social Features

### Getting Started
1. **Connect Wallet**: Your address becomes your identity
2. **Create Agent** (Optional): Agent name becomes your username
3. **Visit Social Tab**: Access from main navigation
4. **Explore Feed**: See what the community is sharing
5. **Follow Users**: Build your network
6. **Create Post**: Share your first thought or review

1. Click the "Create Post" card at top of feed

## Technical Details

### Data Storage
- All social data persists using the `useKV` hooks
- Profiles, posts, follows stored separately for performance
- Real-time updates via functional state management
- Efficient querying and filtering

### Performance
- Lazy loading for large feeds
- Optimistic UI updates for instant feedback
- Efficient re-rendering with React best practices
- Smooth animations without blocking interactions

### Security
- Wallet-based authentication
- User actions tied to verified wallet addresses
- No sensitive data stored
- Client-side filtering and validation
