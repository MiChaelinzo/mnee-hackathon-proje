# Nova AI Marketplace

> **Generative AI-Powered Autonomous Commerce Platform**  
> Built for the [Amazon Nova AI Hackathon](https://amazon-nova.devpost.com/) — Powered by Amazon Nova on AWS

A generative AI marketplace where AI agents can autonomously discover, purchase, and sell services — powered by **Amazon Nova** foundation models on AWS. Features real-time AI recommendations, autonomous agent commerce, smart bundle suggestions, and predictive analytics, all driven by Nova AI capabilities.

## 🧠 Amazon Nova AI Integration

This project is built as a generative AI application using **Amazon Nova on AWS**:

- **Nova Foundation Models** — Core AI engine powering agent intelligence, service recommendations, bundle optimization, and predictive analytics
- **Nova AI Agents** — Autonomous agents that discover, evaluate, and purchase services using AI-driven decision making
- **Nova-Powered Recommendations** — LLM-powered bundle suggestions and trend analysis using Nova models
- **Generative AI Commerce** — AI-generated service descriptions, pricing intelligence, and marketplace insights

> **Hackathon Focus Area**: AI-powered autonomous commerce platform demonstrating Nova foundation model capabilities for intelligent agent-to-agent transactions and generative AI marketplace features.

## 🌟 Key Features

- 🤖 **AI Agent Management** — Create and configure autonomous agents with spending limits
- 🛒 **Service Marketplace** — 15+ AI services across 6 categories
- 📦 **Smart Bundles** — Pre-built and custom service bundles with AI-optimized discounts
- 🧠 **Nova AI Recommendations** — Foundation model-powered bundle suggestions
- 📊 **Trend Analytics** — Historical recommendation tracking with export
- 💰 **Web3 Integration** — Wallet connection with live Nova token balances
- 🔗 **Blockchain Connectivity** — On-chain transaction support
- 📱 **Transaction Explorer** — Real-time transaction feed
- 🔮 **Predictive Analytics** — AI-driven market predictions and insights
- 💬 **AI Chat System** — Nova-powered marketplace assistant

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- [MetaMask Browser Extension](https://metamask.io/download/) (for Web3 features)
- AWS account with Amazon Nova access (for AI features)

### Getting Started
1. Clone the repository and install dependencies
2. Configure your AWS credentials for Amazon Nova access
3. Run `npm run dev` to start the development server
4. Connect your wallet and start exploring AI-powered services
5. Create AI agents and let Nova power autonomous purchases

## 🛠️ Technology Stack

- **AI/ML**: Amazon Nova Foundation Models (Nova 2 Lite, Nova 2 Sonic) on AWS
- **Frontend**: React 19 + TypeScript
- **Styling**: Tailwind CSS v4 + Shadcn v4 Components
- **Web3**: Ethers.js v6 (blockchain integration)
- **Animation**: Framer Motion
- **Charts**: Recharts
- **Build**: Vite
